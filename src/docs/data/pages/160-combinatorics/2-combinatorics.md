# Combinatorics

```js
import {
    permutationsWithRepetition,
    permutationsWithoutRepetition,
    combinationsWithoutRepetition,
    combinationsWithRepetition,
} from 'mz-math';

const res1 = permutationsWithRepetition(3, 2); // 9
const res2 = permutationsWithoutRepetition(4, 4); // 24
const res3 = combinationsWithoutRepetition(16, 3); // 560
const res4 = combinationsWithRepetition(5, 3); // 35
```
